Data dalam Power BI

Data adalah sekumpulan keterangan atau fakta yang diperoleh melalui pengamatan, pencarian, atau pengukuran dari sumber-sumber tertentu. Keterangan-keterangan ini dapat berupa kata-kata, kalimat, simbol, angka, citra, atau properti lainnya.

Dataset dalam Power BI

Power BI merupakan aplikasi yang bertujuan untuk menampilkan informasi secara visual dari suatu dataset. Dataset yang mendasari visualisasi dan laporan di Power BI berasal dari kumpulan data yang diperoleh dari data source.

Terdapat tujuh (7) kategori konektivitas dataset dalam Power BI yang dapat digunakan oleh pengguna dengan lebih dari 500 konektor, diantaranya yaitu:

  1. Flat File

    2. Flat file merupakan sarana penyimpanan sederhana untuk data tabel tetapi tidak memuat hubungan antar tabel di dalamnya. Flat file mencakup file Excel, teks/CSV, PDF, file JSON, dan lain-lain. Dataset pada Flat file tersebut diimpor ke dalam kumpulan data Power BI. Mengimpor data berarti data disimpan dalam file Power BI dan dipublikasikan bersama dengan report Power BI.

    Note:
    Pendekatan Flat file mungkin tidak cocok untuk semua organisasi atau ketika pengguna memiliki volume data yang besar.

  2. Database

    3. Database merupakan kumpulan data yang dikelola sedemikian rupa berdasarkan ketentuan tertentu yang saling berhubungan sehingga mudah dalam pengelolaannya. Beberapa database yang cukup terkenal adalah SQL Server, MySQL, OracleDB, PostgreSQL, IBM, dan lain-lain. Data yang disimpan dalam database biasanya besar dan mungkin memerlukan waktu yang sangat lama untuk dimuat saat diimpor ke Power BI.

    Power BI mengatasi masalah waktu yang lama untuk memuat data dengan menggunakan mode penyimpanan DirectQuery, yang memungkinkan pengguna untuk melakukan kueri data langsung dari sumber data tanpa mengimpor salinan ke dalam Power BI. DirectQuery berguna karena memastikan pengguna selalu melihat versi data yang paling baru (terkini).

  3. Microsoft Fabric

    4. Microsoft Fabric adalah sebuah produk yang terintegrasi dengan Power BI dan merupakan platform analitik yang berfokus pada pengalaman pengguna. Microsoft Fabric memungkinkan pengguna untuk menggabungkan semua data dan analitik organisasi dalam satu tempat.

    Microsoft Fabric terdiri dari beberapa teknologi, seperti Power BI dataset, Dataflows, Datamarts, Warehouses, Lakehouses, KQL Database, yang digabungkan menjadi satu platform SaaS yang terpadu.

  4. Power Platform

    5. Power Platform pada Power BI berfungsi untuk menggabungkan data dari berbagai sumber dan membuat aplikasi bisnis yang dapat diintegrasikan dengan Power BI. Dengan Power Platform, pengguna dapat menggabungkan data dari berbagai sumber dan membuat aplikasi bisnis yang dapat diintegrasikan dengan Power BI. Hal ini memungkinkan pengguna untuk membuat keputusan bisnis yang lebih baik dan lebih cepat

  5. Azure

    6. Azure adalah adalah layanan cloud yang digunakan untuk memproses dan mengelola data. Azure dan Power BI dapat terintegrasi bersama dalam pemrosesan data hingga analitik dan laporan.

    Azure Synapse adalah layanan analitik enterprise yang menggabungkan teknologi SQL untuk data warehousing, teknologi Spark untuk big data, serta fitur Data Explorer untuk analitik log dan time series. Selain itu, Azure Synapse memiliki fitur Pipelines yang memungkinkan integrasi data dan proses ETL/ELT.

    Power BI merupakan salah satu layanan yang sudah secara sudah secara alami terintegrasi dengan Azure Synapse sehingga memungkinkan pengguna untuk mempercepat kinerja dengan solusi bisnis dan manajemen data yang efisien.

  6. Online Services

    7. Power BI memberikan fitur bagi pengguna untuk mendapatkan data dari layanan online lain dan memanfaatkannya dalam laporan dan visualisasi. Menggunakan data dari layanan online mirip dengan menggunakan flat file, namun data ini dapat diperbarui langsung dari sumber layanan online tersebut sehingga memerlukan kredensial untuk terhubung. Layanan online ini mencakup Google Drive, SharePoint Online, Salesforce.

  7. Other (Layanan Lain)

    8. Pada Power BI terdapat fitur bagi pengguna untuk mendapatkan data dari platform atau program lain seperti mengintegrasikannya dengan Python dan R, menggunakan layanan cloud lain, hingga mendapatkan data dari website menggunakan Uniform Resource Locator (URL) pada website tersebut.

Mengenal Power Query

Data yang diimpor pada power BI akan diproses pada Power Query sebelum masuk pada tahap analisis dan visualisasi data. Power Query adalah alat bisnis yang tersedia di Excel dan Power BI yang memungkinkan pengguna mengimpor data dari berbagai sumber, lalu membersihkan, mentransformasi, dan menyusun ulang data sesuai kebutuhan.

Power Query memiliki banyak fitur untuk membantu pengguna membersihkan dan menyiapkan data untuk analisis. Power Query adalah mesin transformasi data dan persiapan data.

Power Query dilengkapi dengan antarmuka grafis untuk mengambil data dari sumber dan Power Query Editor untuk menerapkan transformasi. Dengan menggunakan Power Query, pengguna dapat melakukan pemrosesan ekstraksi, transformasi, dan pemuatan (ETL) data.

Mendapatkan Data dalam Power BI

Pada course ini akan dijelaskan cara menghubungkan power BI melalui Excel, Website, dan Spreadsheet. Sebelum menghubungkan data pada Power BI unduh data source Superstore yang akan digunakan di modul ini.

Data Super store merupakan salah satu open dataset yang tersedia dan memuat informasi tentang penjualan & keuntungan Superstore.

Menghubungkan File Excel

Untuk menghubungkan dataset yang telah diunduh sebelumnya ke dalam Power BI dapat menggunakan langkah langkah berikut di bawah ini:

  1. Buka program Power BI Desktop yang telah terinstal.
  2. Pada program yang terbuka pilih “Excel workbook” pada bagian “Select a data source or start with a blank report” karena akan digunakan format file Excel untuk dataset seperti ilustrasi di bawah ini.
  3. Setelah memilih tautan "Microsoft Excel", jendela "Open" akan terbuka, pilih lembar kerja/file yang sesuai untuk digunakan dalam Power BI seperti ilustrasi di bawah ini.
  4. Setelah file terhubung pada Power BI, akan muncul seperti dibawah ini, atau disebut Window Navigator untuk memilih dataset yang akan digunakan.
  5. Pilih dataset yang akan digunakan pada window Navigator, lalu pilih “Load”, dan akan muncul Window Processing Queries yang menampilkan status transformasi dan menghubungkan data seperti ilustrasi di bawah ini.

    6. Berikut adalah video langkah-langkah praktik keseluruhan menghubungkan file Excel pada Power BI.

Menghubungkan Data dari Website

Selain menggunakan data dari file Excel, Power BI dapat menggunakan data dari website menggunakan Uniform Resource Locator (URL). Berikut adalah langkah-langkah untuk menghubungkan file dari website:

  1. Buka program Power BI Desktop yang telah terinstal.
  2. Pada program yang terbuka pilih “Get data from other sources” pada bagian “Select a data source or start with a blank report” karena akan digunakan format lain untuk dataset seperti ilustrasi di bawah ini.
  3. Pilih menu Other pada window Get Data, lalu pilih website untuk menghubungkan data dari website, dan pilih connect.
  4. Akan muncul window From Web, lalu masukkan URL website yang berisi dataset yang akan dihubungkan pada Power BI. Pada modul ini akan dicontohkan untuk data populasi pada Worldometer dengan URL https://www.worldometers.info/world-population/population-by-country/
  5. Pilih OK setelah memasukkan URL yang sesuai.
  6. Selanjutnya akan muncul window Connecting yang memuat status Power BI sedang menghubungkan dengan data pada Website.
  7. Setelah data terhubung pada Power BI, akan muncul Window Navigator untuk memilih dataset yang akan digunakan seperti ilustrasi di bawah ini.
  8. Akan muncul Window load yang menampilkan status transformasi dan menghubungkan data.
  9. Dataset yang sudah terhubung akan muncul pada Window Data, dan terdapat informasi mengenai kolom yang terdapat pada dataset yang digunakan.

    10. Berikut adalah video langkah-langkah praktik keseluruhan menghubungkan data dari website pada Power BI.

Menghubungkan Data dari Google Sheets

Selain menggunakan data dari file Excel dan Website, Power BI dapat menggunakan data dari Google Sheet.

Note:
Format file tidak boleh dalam bentuk .csv, .xlsx, .xls tetapi harus dalam spreadsheet google.

Berikut adalah langkah-langkah untuk menghubungkan file dari Google Sheet:

  1. Buka program Power BI Desktop yang telah terinstal.
  2. Pada program yang terbuka pilih “Get data from other sources” pada bagian “Select a data source or start with a blank report” karena akan digunakan format lain untuk dataset seperti ilustrasi di bawah ini.
  3. Pilih menu Get Data, lalu pada bar pencarian ketikkan Google Sheets, dan pilih Google Sheets untuk menghubungkan data dari Google Spreadsheet, dan pilih connect.
  4. Masukkan url dataset Global Superstore terlampir pada window Google Sheet yang muncul lalu pilih ok seperti ilustrasi di bawah.
  5. Sign in ke akun Google dengan memilih “Sign In” apabila belum masuk ke akun Google.
  6. Pilih Connect apabila sudah masuk ke akun google masing-masing.
  7. Pilih dataset yang akan digunakan pada window Navigator, lalu pilih “Load”, dan akan muncul Window Load yang menampilkan status transformasi dan menghubungkan data.
  8. Dataset yang sudah terhubung akan muncul pada Window Data, dan informasi mengenai kolom dataset dapat dilihat dengan mengklik masing-masing dataset tersebut.

Permasalahan yang Sering Muncul dalam Proses Import Data

Ketika menghubungkan data pada power BI terkadang terdapat beberapa permasalahan yang cukup sering muncul. Beberapa permasalah tersebut adalah:

  1. Time-out expired: Permasalahan ini muncul ketika menggunakan database yang cukup besar sehingga memerlukan waktu lebih lama untuk menghubungkan data. Untuk mengatasinya dapat beralih ke versi berbayar untuk menambah batas waktu.
  2. The data format is not valid: Permasalahan ini muncul ketika pengguna mengimpor format data yang tidak valid atau tidak sesuai pada Power BI. Untuk mengatasinya dapat memperbaiki file data terlebih dahulu sebelum dihubungkan pada Power BI.
  3. Missing data files: Permasalahan ini muncul ketika file yang digunakan pada power BI tidak dapat ditemukan, umumnya dikarenakan berpindah directory atau file hilang. Untuk mengatasinya dapat mengatur ulang direktori yang digunakan dan memastikan file tersedia.
